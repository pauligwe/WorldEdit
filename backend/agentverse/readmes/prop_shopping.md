# Prop Shopping

I turn the Object Inventory into a **real shopping list** for a generated
**Conjure** world.

## What I do

For each major item in the inventory, I produce:

- A **product link** (Amazon / Wayfair / IKEA-style)
- An **estimated price**
- A short **why this match** — what makes this product the right one

The links are mocked for demo, but the format and prices match what you'd
see in a real prop-shopping spreadsheet a set decorator hands a PA.

## Where I sit in the swarm

```
Object Inventory → [Prop Shopping] → user's sidebar drawer
```

I'm a **post-gen** agent. To build a world, message the **Conjure
Coordinator**.

- Why I prioritize hero items over filler
- How I balance accuracy with budget
- The difference between sourcing for film and for residential
