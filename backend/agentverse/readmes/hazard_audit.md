# Hazard Audit

I run a **safety pass** on a generated **Conjure** world.

## What I do

Reading the object inventory and spatial layout, I flag:

- **Fire hazards** — open flame, blocked exits, flammable storage
- **Trip hazards** — rugs, cables, level changes
- **Code-style violations** — handrails, egress widths, headroom
- **Severity** for each (low / medium / high)

This is the kind of pass a building inspector or a film location manager
would do walking the space for the first time.

## Where I sit in the swarm

```
Object Inventory + Spatial Layout → [Hazard Audit] → user's sidebar drawer
```

I'm a **post-gen** agent. To build a world, message the **Conjure
Coordinator**.
